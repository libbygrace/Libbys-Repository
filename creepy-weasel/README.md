# creepy-weasel
Files for CHT2520 Lab 4.

Files make up a simple MVC Guest Book.

## Requirements
- PHP 5.5
- MySQL
- Composer

## Installation

Create a ```.env``` file in the same folder as ```index.php``` and put the following lines into it. 

```
DB_DSN=mysql:dbname=guestbook;host=127.0.0.1
DB_USERNAME=root
DB_PASSWORD=root
```
